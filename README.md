# CRM de Ventas

Proyecto base de CRM de ventas para gestionar usuarios y clientes.

## Características

- Autenticación simple de usuarios
- Panel de administración de usuarios
- Gestión de clientes con correo y teléfono
- Configuración de visibilidad de datos por usuario

## Instrucciones

1. Instala dependencias:
   ```bash
   npm install
   ```
2. Ejecuta en desarrollo:
   ```bash
   npm run dev
   ```
3. Abre `http://localhost:3000`.
